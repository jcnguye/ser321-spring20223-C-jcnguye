You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /github -- grabs repo of user -- ip/github?query=users/jcnguye/repos
  • /multiply -- multiply two numbers -- ip/multiply?num1=7&num2=10
  • /roulette -- pick number between 1 - 6 -- ip/roulette?num1=4&num2=6
  • /joke -- put your first and last name -- ip/joke?firstname=jaun&lastname=carlos
